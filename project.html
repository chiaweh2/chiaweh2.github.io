Project



Highlights


  1. Used Monte Carlo error analysis to show the first detection of sea level fingerprint by using NASA GRACE satellite data. (Highlighted by Nature , NASA/JPL , AGU and UCI )

  2. Optimized the spatial and temporal sea level data by integrating various spatially sparse in-situ dataset with globally covered remote sensing dataset through spatial kriging (Gaussian process regression).

  3. Proficiency with various remote sensing data including GRACE, Altimetry, Landsat satellite imagery for extracting the useful physical signals like water/ice mass changes in land hydrology, cryosphere, and ocean.

  4. Applied machine learning technique and established an artificial neural network for a classification problem (using logistic regression) which recognizes the unique spatial pattern in the atmospheric circulation.

  5. Optimized and tested the simulation of mesoscale atmospheric convection by combining different numerical methods and by introducing parallel computing.

  6. Classified the hurricane/typhoon based on rainfall events and tracks in the southwest Pacific area in the past 40 years and further project the possible future changes through multivariate regression.
